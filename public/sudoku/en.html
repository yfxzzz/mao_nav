What is Sudoku?

Sudoku is a logic-based number puzzle that originated in 18th-century Switzerland. The objective is to fill a 9×9 grid so that each column, each row, and each of the nine 3×3 boxes contains the digits 1 to 9, with no repeats.

Rules

  • Fill the 9×9 grid with digits 1-9
  • Each row must contain all digits 1-9 without repetition
  • Each column must contain all digits 1-9 without repetition
  • Each 3×3 box must contain all digits 1-9 without repetition
  • Given digits cannot be changed; you have 3 mistake allowances

Tips

Only Choice: If eight digits already appear in a row/column/box, the remaining cell must be the missing digit.

Elimination: If a digit already exists in a row/column/box, remove it from the candidate lists of other cells in that unit.

Candidate List: Write small numbers in cells and cross them out as they become impossible.

Cross-Hatching: Use row and column intersections to eliminate candidates.

Difficulty

Four levels are provided:

  • Easy: Plenty of given numbers, great for beginners
  • Medium: Moderate givens, some logical reasoning required
  • Hard: Few givens, deeper strategies needed
  • Expert: Minimal givens, extremely challenging
