Game Rules

2048 is a simple yet challenging number puzzle game. Your goal is to merge tiles with the same number to eventually create a tile with the number 2048.

Basic Gameplay

  • Use arrow keys (up, down, left, right) or swipe the screen to move all tiles
  • When two tiles with the same number collide, they merge into one tile with their sum
  • After each move, a new tile (either 2 or 4) appears randomly in an empty spot
  • You win when you successfully create a tile with the number 2048!
  • The game ends when there are no more available moves

Controls

  • Computer: Use arrow keys (‚Üë‚Üì‚Üê‚Üí) or WASD keys to move tiles
  • Mobile devices: Swipe your finger across the screen to move tiles
  • Function buttons:
    • New Game: Reset the game and start a new session
    • Pause: Temporarily stop the game

Game Tips

  • Try to keep your highest number tile in a corner to create more space for merging other tiles
  • Try to keep tiles in sequential order, forming an increasing or decreasing sequence
  • Avoid moving tiles in all directions frequently, as this may disrupt the tile arrangement
  • Plan your moves ahead and think about possible scenarios a few steps in advance

Scoring Rules

Each time you merge two tiles with the same number, you earn points equal to the sum of those numbers. For example, merging two 2-tiles to create a 4-tile earns you 4 points; merging two 4-tiles to create an 8-tile earns you 8 points, and so on. The game records your highest score and saves it locally, allowing you to challenge your personal best at any time.

Day/Night Mode

The game supports day and night mode switching. Click the moon/sun icon in the top right corner to toggle between light and dark themes to adapt to different lighting environments and personal preferences.

Multi-language Support

You can switch languages at any time through the language selector in the top right corner.
